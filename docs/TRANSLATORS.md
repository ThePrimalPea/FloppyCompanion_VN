# Translation Credits

Thanks to everyone who contributed translations for FloppyCompanion's web UI.

## Source Of Truth

Translation contributor data is maintained in:

- `repo/webroot/credits.json` for contributor links and per-language credits
- `repo/webroot/lang/languages.json` for language metadata shown in the UI

The About page reads from those files directly, so they should be updated together when adding a new language.

Want to help translate? See the [Translation Guide](TRANSLATION_GUIDE.md) for how to add or update translations.
